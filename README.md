# 🦦 Ferret Breeding Analytics Dashboard (MVP 3)

A React-based analytics dashboard designed for ferret breeders to track breeding performance, litter outcomes, kit growth, and breeding program metrics.

This project is the third milestone in a multi-stage breeding management software portfolio series:

* MVP 1: Backend CRUD application using Flask, SQLAlchemy, and MySQL
* MVP 2: React breeding dashboard and breeder records
* MVP 3: Analytics, visualization, and litter performance tracking

# 🎯 Project Goals

* React component architecture
* State management with useState
* Data visualization using Recharts
* Dynamic forms
* Conditional rendering
* Dashboard design
* Analytics-focused user interfaces

The project also serves as the foundation for a future full-scale breeding management platform designed specifically for ferret breeding programs, but with intention to flex for other niche species breeding programs.

# 📚 Lessons Learned

This MVP focused heavily on frontend architecture and data visualization.

Major concepts practiced included:

### React State Management

* useState
* Controlled inputs
* Derived state

### Component Design

* Reusable components
* Props
* Parent-child communication
* Conditional rendering

### Data Visualization

* Line charts
* Analytics dashboards
* Dynamic chart generation

### User Experience

* Toggleable analytics panels
* Open/close detail views
* Dynamic filtering
* Responsive layouts

### Problem Solving

Learning how to break large features into smaller reusable components.
Optimizing for UX according to the anticipated needs of the clientele. 

Examples include:

* LitterAverageWeightTracker
* IndividualKitWeightTracker
* LitterNotes
* Analytics chart components
* Dashboard statistic components
* Recombination and reorganization of charts, graphs, and statistics to appropriate pages

This resulted in cleaner code, improved maintainability, and improved UX.
